Single-click self-hosting is coming soon! 

For now, please include the required by environment variables found in [env.ts](https://github.com/crbnos/carbon/blob/main/packages/auth/src/config/env.ts), and setup the following infrastructure:

- [Supabase](https://supabase.com)
- [Upstash](https://upstash.com)
- [Trigger.dev](https://trigger.dev)

If you need help self-hosting Carbon in a GovCloud environment, please [contact us](https://carbon.ms/sales).
